# Trainly - Фитнес и здоровье
Trainly - это веб-приложение для управления фитнес-тренировками, отслеживания прогресса и планирования тренировочных программ. Приложение предлагает персонализированный подход к фитнесу с интуитивно понятным интерфейсом.

## Основные возможности
### Пользовательский функционал
Личный профиль - управление личными данными и настройками

Мои тренировки - создание и управление индивидуальными тренировками

Планы тренировок - разработка и отслеживание тренировочных программ

Выполнение тренировок - пошаговое руководство во время тренировок

Прогресс - отслеживание статистики и достижений

Библиотека упражнений - доступ к базе упражнений с фильтрацией по группам мышц и оборудованию

### Особенности
Тёмная/светлая тема - переключение между темами оформления

Адаптивный дизайн - корректное отображение на всех устройствах

Интуитивный интерфейс - современный UX/UI дизайн

Персонализация - создание собственных упражнений и тренировок

Отслеживание прогресса - визуализация достижений и статистики

## Технологический стек
### Frontend
Vue.js 3 - основной фреймворк

Vue Router - маршрутизация

CSS3 с переменными - стилизация и темы

Font Awesome - иконки

Axios - HTTP-клиент

### Архитектура
Компонентный подход

Модульная структура

SPA (Single Page Application)

## Установка и запуск
### Предварительные требования
Node.js (версия 14.x или выше)

npm или yarn

Запустите приложение в режиме разработки:

bash
npm run serve
Для сборки продакшн-версии:

bash
npm run build
## Основные маршруты:
/ - Главная страница

/login - Страница входа

/registration - Регистрация

/profile/:id - Профиль пользователя

/my-workouts - Мои тренировки

/my-workout-plans - Мои планы тренировок

/workout-plans/new - Создание плана тренировок

/exercises/new - Создание упражнения

/workouts/execute/:id - Выполнение тренировки

/progress - Прогресс и статистика

/about - О приложении

## Аутентификация
Приложение использует JWT-токены для аутентификации. Токен сохраняется в localStorage и автоматически добавляется к каждому запросу через axios-интерсепторы.

## Темы оформления
Приложение поддерживает две темы:

Тёмная тема (по умолчанию) - для комфортной работы в условиях низкой освещённости

Светлая тема - классическое светлое оформление

Тема сохраняется в localStorage и применяется при повторном посещении.

## Адаптивный дизайн
Приложение полностью адаптировано для:

Десктопных устройств (> 1200px)

Планшетов (768px - 1200px)

Мобильных устройств (< 768px)

## API интеграция
Приложение взаимодействует с REST API. Основные эндпоинты:

/login - аутентификация

/logout - выход из системы

/exercises - управление упражнениями

/workouts - управление тренировками

/workout-plans - управление планами тренировок

/muscle-groups - группы мышц

/equipment-types - типы оборудования

## Роли пользователей
Пользователь - полный доступ к личному кабинету, тренировкам и прогрессу

Тренер - расширенные возможности для создания и публикации тренировок

## Конфигурация
Основные настройки приложения находятся в:

vue.config.js - конфигурация Vue CLI

public/index.html - HTML шаблон

src/api.js - настройки API
